# Carpeta de Imágenes

## Instrucciones para agregar tus imágenes:

1. Coloca aquí las siguientes imágenes:
   - `logo.png` o `logo.jpg` - Logo de Cinepolis Valtierra
   - `avatar.jpg` - Poster de Avatar
   - `sonic3.jpg` - Poster de Sonic 3
   - `avengers.jpg` - Poster de Avengers
   - `starwars.jpg` - Poster de Star Wars

2. Los formatos aceptados son: .jpg, .jpeg, .png, .webp

3. Recomendación de tamaños:
   - Logo: 200x200px mínimo
   - Posters: 500x750px (proporción 2:3)
